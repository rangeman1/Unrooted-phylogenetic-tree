Unrooted tree creation is possible using presented script Utree.py. Utree.nwk file contains the tree and script NWKRUr.py can draw it for you. File Similarity.jpg presents distances between different countries based on the food preferences. tSNE.jpg file shows another method of multidimentional reduction called t-SNE applied for the same set of data like in Utree.jpg file.
Data_example.txt file shows you format of input into ut.py script. The ut.py script also generates *.nwk file that you can draw as an unrooted radial tree using NWKRUr.py program.
