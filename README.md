Unrooted tree creation is possible using presented script Utree.py. Utree.nwk file contains the tree and script NWKRUr.py can draw it for you. tSNE.jpg file shows another method of multidimentional reduction called t-SNE applied for the same set of data like in fenole.png file.
Data_example.txt file shows you format of the input txt file into ut.py script. The ut.py script also generates *.nwk file that you can draw as an unrooted radial tree using NWKRUr.py program. Last file utscale.py draws nwk files in the form of unrooted tree with scale.
